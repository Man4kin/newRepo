Обработка "Калькуляция производства"

Общее

Обработка "Калькуляция производства" предназначена для определения целесообразности производства готовой продукции путем проведения предварительного расчета ее себестоимости на основе данных о стоимости входящих в состав ингредиентов.

Основные реквизиты формы обработки

Форма обработки "Калькуляция производства" содержит следующие группы реквизитов:

  • Продукция. Данная группа реквизитов предназначена для определения вида продукции и способа расчета себестоимости входящих в ее состав товаров и ингредиентов.
      • Продукция. Готовая продукция, для которой будет производиться расчет себестоимости. Элемент справочника "Номенклатура".
          • Технологическая карта. Документ "Технологическая карта", по которому будет производиться калькуляция. Элемент справочника "Технологическая карта".
          • Ед.изм. Единица измерения готовой продукции. Значение выбирается из справочника "Единицы измерения".
          • Коэффициент. Коэффициент пересчета единицы измерения продукции в базовую. Реквизит заполняется автоматически.
          • Количество. Количество продукции для производства.
      • Способ расчета. Способ расчета себестоимости ингредиентов, входящих в состав продукции:
          • По последним закупочным ценам
          • По текущим остаткам
        • Фирма. Фирма, по которой будет производиться расчет себестоимости в соответствии с выбранным способом расчета. Значение является элементом справочника "Фирмы".
        • Склад. Склад хранения товарных остатков. Значение является элементом справочника "Склады".
      • Категория розн. цены. Розничная категория цен, для которой осуществляется расчет себестоимости.
      • Розничная цена. Розничная цена продукции. Определяется пользователем или заполняется автоматически при указании в соответствующем реквизите наценки на себестоимость.
      • Наценка. Наценка на себестоимость продукции. Определяется пользователем или заполняется автоматически при задании розничной цены продукции.
  • Ингредиенты. Группа реквизитов предназначена для отображения списка товаров и ингредиентов, входящих в состав продукции. Список может быть сформирован как автоматически (при указании технологической карты), так и вручную – подбором товаров из справочника "Номенклатура".
      • Пересчитать себестоимость. Расчет себестоимости товаров в соответствии с выбранным способом расчета.
      • Создать ТК. Создание технологической карты по результатам проведенной калькуляции. В состав продукции будут входить ингредиенты, указанные в табличной части обработки.
