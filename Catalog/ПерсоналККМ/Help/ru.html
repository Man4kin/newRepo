Справочник "Персонал ККМ"

Общее

Справочник "Персонал ККМ" предназначен для хранения информации о пользователях контрольно-кассовых машин.

Форма списка справочника

Форма списка справочника содержит колонки код и наименование

Форма элемента справочника

При добавлении нового элемента в справочник открывается вспомогательное окно "Персонал ККМ", в котором необходимо заполнить следующие поля:

      • Сотрудник, работающий с ККМ.  Элемент справочника "Сотрудники".
      • Код. Присваивается автоматически.
      • Профиль пользователя ККМ. Элемент справочника "Профили пользователей ККМ".


Также нужно установить флажки, соответствующие данному пользователю:

      • Продавец - если пользователь ККМ является продавцом.
      • Пользователь ККМ - если пользователь ККМ является кассиром.


В группе элементов справочника "Авторизация пользователя ККМ"  нужно ввести пароль пользователя (если он нужен) и выбрать из выпадающего списка тип авторизации (ввод с клавиатуры, штрих-код, магнитная карта).
