Справочник "Единицы измерения"

Общее

Справочник содержит список единиц, в которых ведется учет всех ТМЦ. Справочник "Единицы измерения" является подчиненным справочником по отношению к справочнику "Номенклатура" и вызывается для ввода и редактирования информации при нажатии на кнопку "Перейти" и выборе соответствующей команды меню.

Позиция номенклатуры характеризуется базовой единицей измерения. Кроме базовой, товар может иметь много других единиц измерения. Например, сигареты могут иметь такие единицы измерения: пачка, блок, коробка, контейнер, вагон. Каждая единица измерения товара имеет свой коэффициент пересчета относительно базовой единицы.
Карточка каждого товара содержит список единиц измерения, допустимых для данного товара. Разные товары содержат разные списки единиц измерения. При выписке товаров в документах  можно использовать только те единицы измерения, которые допустимы для данного товара.
В справочнике "Единицы измерений" находятся все возможные единицы измерения для всех товаров, перечисленных в каталоге номенклатуры.

Форма списка справочника

Справочник содержит следующие реквизиты:

      • Наименование.
      • Единица по классификатору.
      • Коэффициент.

Форма элемента справочника

При добавлении нового договора открывается окно "Единицы измерения", в котором заполняются следующие поля:

  • Наименование. Наименование единицы измерения, заполняется вручную.
  • Единица по классификатору. Определяется соответствие данной единицы по отношению к справочнику "Классификатор единиц измерения", где указывается полное наименование выбранной единицы измерения.
  • Коэффициент  пересчета единицы измерения товара в базовую.
