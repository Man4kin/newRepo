Справочник "Фирмы"

Общее

Справочник содержит сведения о  фирмах, от лица которых ведется учет.
В Конфигурации предусмотрена возможность ведения учета хозяйственных операций от лица компании, состоящей из нескольких фирм, причем отчетность может быть получена как для каждой отдельной фирмы, так и сводная для всей компании.
Данные из этого справочника используются почти во всех документах.

Форма списка справочника

Форма списка справочника "Фирмы" содержит следующие данные:

        • Код
        • Наименование
        • Префикс
        • Полное наименование
        • Основной банковский счет

Форма элемента справочника

При добавлении нового элемента открывается окно "Фирмы" .

В данном окне необходимо заполнить следующие реквизиты:

        • Наименование фирмы.
        • Код. Присваивается автоматически.
        • Полное наименование.  Поле для размещения полного официального наименования организации
        • Место ведения БУ. Элемент справочника "Структурные единицы". Пользователем указывается структурная единица, в которой будет вестись бухгалтерский учет.
        • Префикс. Устанавливается префикс - три буквенных символа в начала каждого номера документов данной организации.

Окно содержит вкладки:

        • Налоговые данные
        • Коды
        • Центр доходов и расходов
        • Адреса и ответственные лица
        • Учетная политика
        • Банковские счета
        • Дополнительные свойства
        • Подразделения

Вкладка  "Налоговые данные"

На данной вкладке нужно вручную заполнить поле "Наименование налогоплательщика при перечислении налогов" (полное юридическое наименование организации для отражения в платежных документах).

Вкладка включает в себя группы элементов справочника:

          • Реквизиты
            • ИНН
            • КПП
            • Дата регистрации
            • ОГРН
          • ИФНС
            • Код ИФНС
            • Наименование  ИФНС

Вкладка  "Коды"

Вкладка включает в себя следующие группы элементов справочника:

          • Коды по ОКАТО и ОКПО. В группе заполняются следующие реквизиты:
            • Код по ОКАТО
            • Код по ОКПО
          • Организационно-правовая форма по ОКОПФ. В группе заполняются следующие реквизиты:
            • Код
            • Наименование
          • Форма собственности по ОКФС. В группе заполняются следующие реквизиты:
            • Код
            • Наименование
          • Вид деятельности по ОКВЭД. В группе заполняются следующие реквизиты:
            • Код
            • Наименование

Важно! Данные вкладки используются для заполнения регламентированной отчетности в ГНИ (различные формы бухгалтерского баланса).

Вкладка  "Центры доходов и расходов"

На данной вкладке заполняются следующие реквизиты:

          • Склад. Заполняется из списка складов.
          • Вид номенклатуры. Элемент справочника "Виды номенклатуры". Например, продажа товар, оказание услуг и т.д.
          • Подразделение. Элемент справочника "Подразделения".
          • Доход: отношение к ЕНВД. Заполняется из выпадающего списка: участвует в ЕНВД, не участвует в ЕНВД.
          • Расход: отношение к ЕНВД. Заполняется из выпадающего списка: участвует в ЕНВД-облагаемой деятельности; не участвует в ЕНВД-облагаемой деятельности; доля участия пропорциональна реализованной стоимости.

Важно! Данные вкладки используются для корректного формирования итогового финансового результата Вашей фирмы в том случае, если часть ее деятельности попадает под ЕНВД облагаемую деятельность. Также  данные вкладки используются для аналитики по счетам бухгалтерского учета 44, 90, 91.

Вкладка "Адреса и ответственные лица"

Вкладка содержит колонки:

          • Параметр. В этой колонке находится список параметров для заполнения пользователем:
            • Юридический адрес. Заполняется вручную.
            • Почтовый адрес. Заполняется вручную.
            • Руководитель. Тип значения выбирается из списка. Если тип - строка, то данные вводятся вручную, если тип сотрудники, то данные заполняются из справочника "Сотрудники".
            • Главный бухгалтер.
            • Кассир.
            • Телефон. Заполняется вручную.
          • Значение параметра. Заполняется вручную.

Вкладка "Учетная политика"

Вкладка содержит список параметров бухгалтерского и налогового учета, значения которых пользователь заполняет из выпадающего списка, либо устанавливает соответствующие флажки.

Вкладка "Банковские счета"

Табличная часть вкладки содержит  две колонки:

          • Наименование
          • Банк

При добавлении нового счета открывается окно "Банковские счета" (элемент справочника "Банковские счета").

Вкладка  "Дополнительные свойства"

Вкладка заполняется с помощью справочников "Свойства объектов"  и "Значения свойств" .

Вкладка "Подразделения"

В табличной части вкладки необходимо заполнить колонку наименование

При добавлении нового элемента открывается окно "Подразделения" . Элемент справочника "Подразделения", содержит список штатных подразделений фирмы.
