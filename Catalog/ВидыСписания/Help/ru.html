Справочник "Виды Списания"

Общее

Данный справочник предназначен для учета видов списаний товаров в процессе хозяйственной деятельности предприятия.
Впоследствии существует возможность получить отчет по суммам списаний в разрезе видов списаний.

Форма списка справочника

Форма списка справочника "Виды списания" содержит следующие колонки:

        • Наименование. Наименование вида списания товара, заполняется вручную.
        • Счет отнесения потерь. Заполняется с помощью плана счетов бухгалтерского учета.
        • Включать НДС в потери. Для включения суммы НДС в потери данный флажок должен быть установлен.

Редактирование и ввод новых элементов производится непосредственно в форме списка справочника.
