Справочник "Контрагенты"

Общее

В данном справочнике хранится информация о контрагентах.
Контрагенты – это общее понятие, куда включены поставщики и покупатели. Это могут быть как организации, так и частные лица. В качестве контрагента может быть компания, имеющая несколько собственных юридических лиц. В этом случае в справочник "Контрагенты" заносится группа,  например, «Компания», и все юридические лица, входящие в состав компании заносятся в эту группу.

ВАЖНО! Вводя данные о контрагенте, следует помнить, что от полноты введенной информации зависит правильность заполнения печатной формы различных документов (приходных, расходных, платежных).

Сведения, которые необходимо ввести о клиенте, объединены в несколько групп, каждая из которых расположена на соответствующей вкладке диалога.
Контрагенты могут быть сгруппированы по группам классификатора справочника, например, "Поставщики", "Покупатели" и "Реализаторы".

Форма списка справочника

Форма списка справочника "Контрагенты" содержит следующие данные:

          • Код
          • Наименование
          • ИНН
          • Полное наименование

Форма элемента справочника

При добавлении нового элемента в справочник открывается вспомогательное окно "Контрагенты", в котором необходимо заполнить следующие реквизиты:

          • Наименование. Краткое название контрагента. Наименование должно быть таким, чтобы его было легко найти  с помощью быстрого поиска. Например: Иванов ЧП - это наименование, а ЧП "Иванов свидетельство 7777" - полное наименование.
          • Код. Присваивается автоматически.

Окно содержит следующие вкладки:

          • Основная
          • Контактная информация
          • Банковские счета
          • Договоры
          • Параметры заказа
          • Дополнительные свойства
          • Адреса и ответственные лица


Вкладка "Основная"

На данной вкладке сначала нужно заполнить поле тип контрагента (статус контрагента: юридическое или физическое лицо - выбирается из выпадающего списка.)

В зависимости от типа контрагента поля к заполнению будут выглядеть следующим образом:

          • Юридическое лицо.
            • Полное наименование. Полное наименование контрагента для отражения в печатных формах документации.
            • ИНН. Идентификационный номер контрагента, для отражения в печатных формах документов.
            • КПП. Код поставновки на учет, для отражения в печатных формах документов, заполняется вручную.
            • Код по ОКПО.
            • Организационная форма. Элемент справочника "Организационные формы".  Например, формой может быть: ООО, ЗАО, ПБОЮЛ и т.д.
            • Адрес в Интернете. Информация о контрагенте в интернете: сайт, электронная почта и т.д.
          • Физическое лицо.
            • ФИО. Фамилия, имя и отчество, заполняется вручную. 
            • Документ.  Паспортные данные или данные документа, удостоверяющего личность.
            • ИНН. Идентификационный номер физического лица, для отражения в печатных формах документов, заполняется вручную.
            • Адрес в Интернете. Информация о контрагенте в интернете: сайт, электронная  почта и т.д.

ВАЖНО! После заполнения вкладки "Основные" необходимо нажать кнопку "Сохранить", после чего остальные вкладки станут доступными для редактирования.

Вкладка "Контактная информация"

На вкладке "Контактная информация" можно внести следующие данные о контактных лицах организации-контрагента:

            • Сотрудник (ФИО сотрудника)
            • Должность
            • Телефон
            • E-mail

Вкладка "Банковские счета"

При заполнении вкладки  открывается окно "Банковские счета", которое является элементом справочника "Банковские счета". 
При  вызове этого окна из справочника "Контрагенты" в поле "Корреспондент" будет автоматически указываться название данного контрагента.
После заполнения информации окна  на вкладке "Банковские счета" отразятся следующие данные:

            • Код
            • Наименование
            • Банк

Вкладка "Договоры"

При заполнении вкладки "Договоры" открывается окно "Договоры", которое является элементом справочника "Договоры". 
При  вызове этого окна из справочника "Контрагенты" в поле "Контрагент" будет автоматически указываться название данного контрагента.
После заполнения информации окна на вкладке "Договоры" отразятся следующие данные:

            • Код
            • Наименование
            • Фирма
            • Вид договора
            • Тип договора


Вкладка "Параметры заказа"

На данной вкладке заполняются параметры заказа, произведенного у данного контрагента.
Вкладка содержит следующие реквизиты:

            • Срок поставки заказа, дни. Количество дней, отведенное для поставки заказа.
            • Отправлять заказы для этого контрагента по электронной почте. Флаг устанавливается в случае, если заказы отправляются по e-mail.
            • Шаблон темы. Поле можно заполнить как вручную, так и автоматически - с помощью кнопки "По умолчанию".
            • Шаблон письма. Поле можно заполнить как вручную, так и автоматически - с помощью кнопки "По умолчанию".
            • Контактное лицо. Элемент справочника "Контактные лица".

 

ВАЖНО! Поля "Шаблон темы", "Шаблон письма" и "Контактное лицо" активны только при установленном флаге "Отправлять заказы для этого контрагента по электронной почте".

Вкладка "Дополнительные свойства"

Вкладка заполняется при помощи справочников "Свойства объектов" и "Значения свойств".

Вкладка "Адреса и ответственные лица"

На  вкладке справочника "Контрагенты" - "Адреса и ответственные лица" заполняются следующая информация о фирме-контрагенте:

        • Юридический адрес
        • Почтовый адрес
        • Руководитель
        • Главный бухгалтер
        • Кассир
        • Телефон

ВАЖНО! Реквизиты данной вкладки должны быть обязательно заполнены. От полноты введенной информации зависит правильность заполнения печатной формы различных документов (приходных, расходных, платежных).

Вкладка "Паспортные данные"

Данная вкладка появляется только если в реквизите "тип контрагента" указано значение "физическое лицо".
Реквизиты данной вкладки очевидны и в комментариях не нуждаются.
