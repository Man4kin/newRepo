Справочник "Номенклатура"

Общее

Этот справочник является многоуровневым и предназначен для хранения номенклатуры:

      • товаров
      • наборов
      • тары
      • собственной продукции
      • услуг

Категории того, что это -  возвратная тара, материалы, продукция определяются в момент поступления номенклатуры на предприятие.
Данные из этого справочника используются в таких документах, как «Приходная накладная», «Расходная накладная» и т.д.

Форма списка справочника

В форме списка справочника содержится следующая информация:

      • Код
      • Артикул
      • Наименование
      • Вид товара
      • Базовая единица измерения

Форма элемента справочника

При добавлении нового элемента открывается вспомогательное окно "Номенклатура".
Сначала необходимо заполнить поле "Наименование" , при этом Код будет присвоен автоматически.
В поле "Родитель" указывается группа товаров, которой будет подчинен данный элемент.

Окно содержит следующие вкладки:

      • Основное
      • Связи
      • Дополнительные свойства и категории
      • Заказы
      • Прочее


В после "Единицы измерения" надо указать базовую единицу измерения из  "Классификатора единиц измерения" .
Флаг "Дробное количество при розничной продаже" ставится, если товар можно продавать в дробном количестве.

Вкладка

"Основное"

На данной вкладке нужно заполнить следующие поля:

            • Артикул. Артикул товара, заполняется вручную.
            • Вид товара. Заполняется из выпадающего списка: товар, набор, тара, ингридиент, услуга.
            • Тип товара. Штучный или весовой, выбирается из списка.
            • Ставка НДС. Численное значение ставки НДС, элемент справочника "Ставки НДС".
            • Страна. Страна, из которой поступает импортный товар. Элемент справочника "Классификатор стран мира".
            • Дополнительный учет. Заполняется из выпадающего списка: не вести, вести учет по характеристикам, вести учет по срокам годности.

В табличной части вкладки, слева, после сохранения элемента автоматически подставляется базовая единица измерения. В дальнейшем здесь указываются дополнительные единицы измерения товара с коэффициентами пересчета в базовую единицу измерения.

В табличной части вкладки, справа, для возможных единиц измерения товара вводится штрих-код товара при помощи сканера штрих-кодов либо вручную.

Вкладка "Связи"

Вкладка содержит группы элементов справочника:

            • Список аналогов. Список товаров, на которые можно произвести замену отсутствующего товара. Элемент справочника "Номенклатура".
            • Список сопутствующих товаров. Осуществляется подбор сопутствующих товаров из списка уже имеющихся товаров в справочнике "Номенклатура". Сопутствующими считаются те товары, которые чаще всего приобретаются одновременно с данным товаром.

Вкладка "Доп. свойства и категории"

Вкладка заполняется при помощи справочников "Свойства объектов" и "Значения свойств".

Вкладка "Заказы"

Вкладка включает в себя две группы элементов справочника:

            • Основные параметры. В группе заполняются следующие реквизиты:
              • Свойство. Колонка содержит основные параметры заказов.
              • Значение. Значения свойств заполняются пользователем из выпадающего списка.
            • Параметры, зависящие от метода.  Вкладка заполняется при помощи справочников "Свойства объектов" и "Значения свойств".

Вкладка "Прочие"

На вкладке "Прочие" заполняются следующие поля:

              • Вид номенклатуры. Элемент справочника "Виды номенклатуры" .
              • Вид материала
              • Норма убыли розница
              • Норма убыли опт.
              • Норма предпродажной подготовки.
