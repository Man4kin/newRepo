Справочник "Классификатор единиц измерения"

Общее

Справочник "Классификатор единиц измерения" содержит полные наименования единиц измерения, в которых ведется учет всех ТМЦ.
В справочник вводится список используемых компанией единиц измерения (штуки, контейнеры, ящики, коробки и т.д.). Они используются при задании единиц измерения позиций номенклатуры.

Форма списка справочника

Формы списка справочника содержит следующие данные:

      • Код
      • Наименование
      • Полное наименование.

Форма элемента справочника

При добавлении нового элемента в справочник открывается вспомогательное окно "Элемент классификатора единиц измерения", в котором заполняются соответствующие поля:

      • Краткое наименование единицы измерения, заполняется пользователем вручную.
      • Код. Присваивается автоматически.
      • Полное наименование единицы измерения, заполняется вручную.

При заполнении справочника можно воспользоваться общероссийским классификатором единиц измерения (ОКЕИ).
Для добавления данных из общероссийского классификатора нужно нажать на кнопку "Подбор из ОКЕИ" и открыть таблицу Общероссийского Классификатора Единиц Измерения. В этой таблице можно осуществлять поиск стандартными средствами, предоставляемыми программой «1С:Предприятие». Дважды щелкнув мышью на выбранной позиции, можно вызвать заполненное окно элемента таблицы и, нажав кнопку ОК или Сохранить, добавить новую единицу измерения в справочник "Классификатор единиц измерения".

Данные из справочника "Классификатор единиц измерения" используются в справочнике "Единицы измерения".
