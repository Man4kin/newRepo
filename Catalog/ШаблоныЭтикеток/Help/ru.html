Шаблоны этикеток принтера штрих-кодов


Справочник содержит шаблоны этикеток для печати на принтере штрих-кодов.

 

Один из шаблонов может быть назначен шаблоном по умолчанию, для этого в шаблоне есть специальный реквизит-флажок - признак шаблона по умолчанию. Этот признак может быть установлен только у одного шаблона. Если Вы устанавливаете этот признак у другого шаблона, то при сохранении объекта данный признак снимается у остальных шаблонов автоматически.

Для добавления поля в шаблон необходимо в табличном документе выделить область и выбрать поле из списка доступных.
