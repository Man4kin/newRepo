Справочник "Функциональные обязанности"

Общее

Данный справочник используется для определения функциональных обязанностей сотрудников организации.
Сотрудник, занимающий одну должность, может одновременно выполнять несколько функциональных обязанностей. Справочник заполнен заранее определенным списком функциональных обязанностей.

Важно! Используется только во встроенном механизме бизнес-процессов 1С:Предприятия 8.0

Форма списка справочника

Форма списка справочника содержит перечень наименований  функциональных обязанностей сотрудников.
Редактирование и ввод новых элементов производится непосредственно в форме списка справочника.
