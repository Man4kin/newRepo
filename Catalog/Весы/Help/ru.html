Справочник "Весы"

Общее

Этот трёхуровневый справочник используется только в том случае, если в ассортиментном перечне товаров присутствуют весовые товары. В нем содержится перечень всех электронных весов фирмы и список весовых товаров, прикрепленных к данным весам.

Форма списка справочника

Форма списка справочника представляет собой окно, разделенное на три части. В левой части окна находится иерархический список всех весов фирмы.

Добавление новых весов осуществляется в средней части окна в колонке "Наименование".
Если курсор в левой части окна установлен на уже существующие весы, то создаваемый элемент будет подчиненным по отношению к этим весам.

В правой части окна составляется список весовых товаров для выбранных весов, при этом заполняются следующие колонки:

      • Номенклатура. Заполняется из списка весовых товаров.
      • Характеристика. Заполняется автоматически при выборе товара из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.

ВАЖНО!  Товары, подлежащие отбору, должны обязательно иметь тип "весовой". Тип товара задается в карточке товара (справочник "Номенклатура").
