Справочник "Кредитные карты"

Общее

Этот справочник содержит список типов кредитных карт.
Для каждой из них необходимо задать код, наименование, выбрать курс валюты и валюту, указать, разрешен ли возврат денежной суммы по данной кредитной карте. При необходимости нужно задать предельную сумму карты – сумму, при превышении которой требуется подтверждение банка.
Данные из этого справочника используются в документе "Z-отчет".

Форма списка справочника

Форма списка справочника содержит следующие данные:

        • Наименование
        • Валюта
        • Лимит суммы

Форма элемента справочника

При добавлении нового элемента в справочник открывается окно "Кредитные карты", в котором необходимо заполнить следующие поля:

        • Наименование. Наименование типа кредитной карты, заполняется вручную. Например, Visa.
        • Валюта. Валюта, в которой ведутся расчеты по кредитной карте. Элемент справочника "Валюты".
        • Тип курса. Из выпадающего списка выбирается нужный курс: наш (внутренний) или курс банка.
        • Лимит суммы. Указывается предельная сумма карты – сумму, при превышении которой требуется подтверждение банка.
        • Флаг возврата. Ставится в случае, если при покупке с использованием данной кредитной карты разрешен возврат товара.

В группе элементов справочника "Для автоматической авторизации" необходимо заполнить следующие данные:

      • Soft - предоставляется банком
      • Ident - предоставляется банком


Табличная часть окна содержит вкладки:

          • Префиксы карт
          • Фирма и банковский счет

Вкладка "Префиксы карт"

На данной вкладке нужно указать префикс кредитной карты (предоставляются банком), порядковый номер присваивается автоматически.

Вкладка "Фирма и банковский счет"

ВАЖНО! Для заполнения данной вкладки элемент необходимо сохранить!

Вкладка содержит следующие поля:

              • Период действия карты. Заполняется с помощью календаря или вручную.
              • Структурная единица. Элемент справочника "Структурные единицы". Например, магазин №1.
              • Фирма. Фирма, от лица которой ведутся расчеты по данной кредитной карте, элемент справочника "Фирмы".
              • Банковский счет. Номер банковского счета фирмы, от лица которой ведутся расчеты по данной кредитной карте, элемент справочника "Банковские счета".
