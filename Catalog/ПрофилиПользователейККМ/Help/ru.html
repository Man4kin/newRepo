Справочник "Профили пользователей ККМ"

Общее

Справочник предназначен для хранения информации о профилях пользователей ККМ. Например, кассир, старший кассир.
Данные из этого справочника используются в справочнике "Персонал ККМ". 

Форма списка справочника

Форма списка справочника содержит код и наименование профиля пользователя ККМ.

Форма элемента справочника

При добавлении нового элемента в справочник открывается окно "Профили пользователей".
В этом окне нужно вручную заполнить поля:

      • Наименование
      • Дополнительная информация
