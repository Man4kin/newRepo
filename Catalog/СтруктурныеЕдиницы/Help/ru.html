Справочник "Структурные единицы"

Общее

Справочник содержит информацию об удаленных структурных подразделениях фирмы: обособленных предприятиях (например, сети розничных магазинов) и других информационных базах.

Форма списка справочника

Форма списка справочника "Структурные единицы" представляет собой окно, разделенное на две части.
В левой части окна  содержится список структурных единиц
При перемещении курсора по списку левой части окна -  в правой части окна отображается элементы, относящие к данной структурной единице.

Правая часть окна справочника содержит следующие вкладки:

      • Склады. Элемент справочника "Склады".
      • ККМ. Элемент справочника "ККМ".
      • Кассы. Элемент справочника "Кассы фирмы".
      • Диапазоны весовых кодов. Элемент справочника "Диапазон весовых кодов".


При перемещении по вкладкам отображается список уже существующих элементов.

При добавлении новых элементов на каждой вкладке открывается  вспомогательное окно, принадлежащее соответствующему справочнику ("Склады", "ККМ"), либо заполняются данные непосредственно в форме списка ("Кассы", "Диапазон весовых кодов").
Таким образом, можно ввести новый элемент в вышеперечисленные справочники непосредственно из формы справочника "Структурные единицы".
