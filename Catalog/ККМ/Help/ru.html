Справочник "ККМ"

Общее

Данный справочник содержит список кассовых комплексов, применяемых при регистрации розничных продаж на Вашем предприятии.
Денежные средства, полученные при продаже товара в розницу (в торговом зале,  розничном магазине и т.д.), поступают в кассу ККМ.
Передача денежных средств из кассы ККМ в основную кассу компании (прием выручки с ККМ) оформляется с помощью документа "Приходный кассовый ордер",

Форма списка справочника

В форме списка справочника "ККМ"  необходимо заполнить поле "Структурная единица" (заполняется из выпадающего списка, например, Центральный офис, Магазин №1 и т.д.).
Форма списка справочника содержит следующие данные:

          • Код
          • Наименование
          • Фирма
          • Тип кассы

Форма элемента справочника

При добавлении нового элемента в справочник открывается окно "ККМ" .

В этом окне необходимо заполнить следующие поля:

          • Наименование. Название или порядковый номер ККМ.
          • Фирма. Название фирмы, которой принадлежит ККМ. Элемент справочника "Фирмы".
          • Код. Присваивается автоматически.
          • Тип ККМ. Указывается, автоматизированная или неавтоматизированная касса. Заполняется выбором из выпадающего списка.
          • Номер регистрационный. Номер кассы, присвоенный налоговым органом. Заполняется вручную.
          • Валюта. Валюта, в которой производятся расчеты на данной ККМ. Элемент справочника "Валюты".
          • Номер производителя. Заводской номер ККМ, заполняется вручную.

Документ содержит вкладки:

          • Отделы
          • Персонал
          • Скидки

Вкладка "Отделы"

На данной вкладке  заполняется поле "Отдел", которое является элементом списка складов. Именно этот отдел (склад) обслуживает данная ККМ.

Вкладка "Персонал"

На этой вкладке заполняется поле "Кассир", которое является элементом справочника "Персонал ККМ". Данная вкладка описывает список пользователей, допущенных для работы на данной ККМ.

Вкладка "Скидки" 

На вкладке "Скидки"  нужно заполнить следующие колонки:

            • Категория цен. Элемент справочника "Категории цен", например, розничная.
            • Дисконтная схема. Элемент справочника "Дисконтные схемы". Именно эта дисконтная схема назначается для указанной категории цен.
