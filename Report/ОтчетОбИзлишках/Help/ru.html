Отчет об излишках

Общее

Отчет предназначен для получения данных о количестве дней, которое осталось до полной распродажи товаров. Ограничивая данные отчета определенным количеством дней остатка, можно получить список товаров, которые за это количество дней не будет распродано, то есть излишки товарных запасов.
