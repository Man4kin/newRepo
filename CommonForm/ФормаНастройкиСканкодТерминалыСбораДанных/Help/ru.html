Форма настройки обработчика драйвера "Терминалы сбора данных"

Форма предназначена для настройки параметров работы терминалов сбора данных. Форма позволяет настраивать следующие параметры устройства:

  • Модель
    Модель подключаемого устройства.
  • Порт
    Порт, к которому подключен терминал сбора данных.
  • Скорость
    Скорость обмена данных с терминалом.
  • ИК - подставка
    В случае, если терминал сбора данных подключен к компьютеру при помощи ИК-подставки, необходимо установить данный флажок.
  • Разделитель базы
    Символ или код символа, используемые в качестве разделителя полей в базе товаров терминала.
  • Разделитель документа
    Символ или код символа, используемые в качестве разделителя полей в документах терминала сбора данных.
  • Номер базы
    Номер базы товаров терминала, в которую происходит выгрузка.
  • Номер документа
    Номер документа терминала сбора данных, из которого происходит загрузка.
  • Выбирать источник загрузки данных
    Если указано, что необходимо выбирать источник данных, то при загрузке из ТСД появится форма выбора. В форме необходимо будет указать, откуда производить загрузку данных: база или документ.
  • Очищать документ после загрузки из ТСД
    Если указано, что необходимо очищать документ при загрузке из ТСД, то после загрузки исходный документ терминала будет очищен. При загрузке из базы терминала база не очищается.

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
