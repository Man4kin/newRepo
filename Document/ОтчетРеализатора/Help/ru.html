Документ "Отчет реализатора"

Общее

Товарный документ "Отчет реализатора" оформляется после получения нашей фирмой товара на реализацию, т.е. в случае, если наша фирма является комиссионером.
Отчет предназначен для составления отчетов о продажах для комитентов.
Документ вводится на основе данных о реализации товара.

Форма шапки документа

При добавлении нового документа открывается окно "Отчет реализатора".
Номер, дата и автор указываются в шапке документа автоматически.


В шапке документа нужно указать следующие реквизиты:
          • Фирма, в которой проводится инвентаризация. Элемент справочника "Фирмы"
          • Тип отчета. Заполняется из выпадающего списка: комитенту или от комиссионера.
          • Контрагент, у которого товар взят на реализацию. Элемент справочника "Контрагенты".
          • Договор с контрагентом. Элемент справочника "Договоры".

Табличная часть документа

Табличная часть документа заполняется при помощи кнопки «Заполнить».  При этом в табличную часть попадают все проданные товары на дату документа, взятые у контрагента на реализацию по договору комиссии.

В табличной части документа содержатся следующие колонки:

          • Номенклатура. Подбор товара осуществляется из справочника "Номенклатура".
          • Характеристика. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
          • Количество. Заполняется вручную при выборе товара.
          • Единица измерения товара. Элемент справочника "Единицы измерения".
          • Коэффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
          • Цена. Заполняется автоматически - как цена товара, принятого на реализацию.
          • Сумма. Заполняется автоматически - как сумма товара, принятого на реализацию.
          • Цена продажи.  Заполняется автоматически по результатам продаж товара.
          • Сумма продажи. Заполняется автоматически.
          • Комиссионное вознаграждение. Рассчитывается автоматически, в соответствии с указанным в договоре с контрагентом способом расчета комиссионного вознаграждения.
          • Сумма НДС комиссионного вознаграждения.
          • Партия товара. Элемент справочника "Партии".
          • По документу. Тип данных выбирается из списка: Z-отчет или расходная накладная. В зависимости от типа данных пользователем выбирается элемент соотвтествующего документа.
