Документ "ПКО"

Общее

Документ «Приходный кассовый ордер» предназначен для отражения факта прихода наличных денежных средств в кассу фирмы.
Приходный кассовый ордер можно ввести на основании расходной накладной.

Форма шапки документа

При добавлении нового документа открывается окно "ПКО".
Номер, дата и автор указываются в шапке документа автоматически.


В шапке документа содержатся следующие реквизиты:

        • Фирма, от лица которой проводятся взаиморасчеты. Элемент справочника "Фирмы".
        • Касса, по которой осуществляется движение наличных денежных средств. Элемент справочника "Кассы фирмы".
        • Сумма, валюта. Численное значение суммы ПКО вводится вручную. Валюту необходимо выбрать из выпадающего списка.

Табличная часть документа

Табличная часть документа содежит вкладки:

        • Аналитика.
        • Расчеты по договору
        • Закрытие обязательств
        • Финансовые операции


Вкладка "Аналитика"

На вкладке "Аналитика" необходимо заполнить следующие реквизиты:

      • классификация операции. Выбирается из выпадающего списка:
          • расчеты с покупателями
          • расчеты с подотчетниками
          • перемещение  Банк -> Касса
          • перемещение Касса -> Касса
          • прочие выплаты и поступления
      • корреспондирующий счет. Заполняется из выпадающего списка.

ВАЖНО! Поле "Корреспондирующий счет" заполняется только для прочих выплат и поступлений. Для всех остальных классификаций операции заполняется табличная часть документа.


При выборе классификации операции (кроме прочих выплат и поступлений) автоматически заполняется колонка "Вид субконто".
Колонка "Значение субконто" заполняется из соответствующих справочников в зависимости от классификации операции:

          • "Контрагенты", "Договоры" - для классификации "расчеты с покупателями"
          • "Сотрудники" - для классификации "расчеты с подотчетниками "
          • "Банковские счета" - для классификации "перемещение  Банк -> Касса "
          • "Кассы фирмы" - для классификации "перемещение  Банк -> Касса "
          • "ККМ "- для классификации "перемещение Касса -> Касса "

 

 Вкладка "Расчеты по договору"

ВАЖНО! Данная вкладка доступна только в том случае,  если на вкладке "Аналитика" в поле "Классификация операции" выбраны "Расчеты с поставщиками" или "Расчеты с покупателями".

На вкладке  необходимо заполнить следующие реквизиты:

      • Правило расчета сумм. Заполняется  из выпадающего списка:
          • Сумма -> Сумма по договору. При изменении суммы документа производится изменение реквизита «Сумма по договору» в соответствии с формулой:  «сумма по договору»= «сумма документа»/«курс валюты договора».При изменении реквизита «Сумма по договору» не производится изменение суммы документа, а производится только изменение  «Курса валюты договора».
          • Сумма <- Сумма по договору. При изменении реквизита «Сумма по договору» производится изменение суммы документа в соответствии с формулой:  «сумма документа»= «сумма по договору»*«курс валюты договора»
            При изменении суммы документа не производится изменение реквизита «Сумма по договору», а производится только изменение  «Курса валюты договора».
          • Суммы не расчитываются. Сумма документа и «Сумма по договору» не изменяются в зависимости от того, какую сумму вы изменяете, а рассчитывается только реквизит «Курс валюты договора».
      • Сумма по договору.  Заполняется вручную.
      • Курс валюты договора.
      • Ставка НДС. Процент ставки НДС указывается из справочника "Ставки НДС".
      • Сумма НДС. Рассчитывается автоматически.

Вкладка "Закрытие обязательств"

Важно! Данная вкладка доступна только в том случае,  если на вкладке "Аналитика" в поле "Классификация операции" выбраны "Расчеты с поставщиками" или "Расчеты с покупателями".

Кнопка «FIFO» осуществляет автоматический отбор обязательств для погашения по одноименному методу  (first in – first out (первый пришел – первый ушел).
Кнопка «Очистить» служит для удаления уже подобранных обязательств в табличную часть вкладки (возможно для повторного отбора).
При помощи кнопки «Подбор» вы можете выбрать нужные обязательства для закрытия. Обязательства можно подбирать с учетом ограничения суммы документа (соответствующий флаг, по умолчанию установлен). В этом случае подобрать обязательства на сумму большую, чем в документе невозможно.
Графа «Погашение обязательства в валюте» вы можете редактировать (в меньшую сторону) для частичного закрытия обязательства.
Если сумма документа превышает сумму подобранных обязательств, то система выдает запрос на изменение суммы документа.

После выбора документов на вкладке автоматически заполнятся следующие реквизиты:

          • По документу
          • Валюта
          • Ставка НДС
          • Сумма валютная
          • Сумма
          • Сумма НДС
