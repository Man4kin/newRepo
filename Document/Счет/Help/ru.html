Документ "Счет"

Общее

Документ «Счет» предназначен для оформления предварительной договоренности с клиентом на поставку товаров или услуг.

Форма шапки документа

При добавлении нового документа открывается окно "Счет".
Номер, дата и автор указываются в шапке документа автоматически.


В шапке документа содержатся следующие реквизиты:

        • Фирма, от лица которой ведутся взаиморасчеты. Элемент справочника "Фирмы"
        • Расчетный счет фирмы. Элемент справочника "Банковские счета".
        • Контрагент, с которым ведутся взаиморасчеты. Элемент справочника "Контрагенты".
        • Договор с контрагентом. Элемент справочника "Договоры".
        • Категория цен для расчетов с данным контрагентом. Элемент справочника "Категории цен".
        • Срок резерва товара. Количество дней резерва, заполняется вручную.

Табличная часть документа

В табличной части документа необходимо заполнить следующие колонки:

        • Номенклатура. Пользователем осуществляется подбор товара из справочника "Номенклатура".
        • Характеристика. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
        • Количество товара. Заполняется вручную.
        • Единица измерения товара. Элемент справочника "Единицы измерения".
        • Коэффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
        • Ставка НДС. Элемент справочника "Ставки НДС".
        • Сумма НДС. Заполняется автоматически.
        • Цена товара. Заполняется вручную.
        • Всего. Общая сумма по позиции товара. Заполняется автоматически.
