Документ "Доверенность"

Общее

Доверенность – один из документов, которые существуют в основном в «бумажной» форме.
Конфигурация просто хранит доверенности в электронном виде.

Форма шапки документа

При добавлении нового документа открывается окно "Доверенность" .
Номер, дата и автор указываются в шапке документа автоматически.

В шапке документа нужно заполнить следующие данные:

 

      • Фирма. Элемент справочника "Фирмы"
      • Банковский счет. Элемент справочника "Банковские счета".
      • Сотрудник. Элемент справочника "Сотрудники".
      • Дата действия. Заполняется вручную или с помощью календаря.
      • Контрагент. Элемент справочника "Контрагенты".
      • Договор. Элемент справочника "Договоры"
      • Получение от. ФИО, от кого получена доверенность, заполняется вручную
      • По документу. Заполняется вручную


В табличной части документа необходимо заполнить следующие колонки:

      • Номенклатура. Элемент справочника "Номенклатура".  Добавить товар можно  с помощью кнопки "Добавить", либо с помощью выбора  в опции "Подбор" подпункта "по справочнику".
      • Характеристика. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
      • Количество. Заполняется вручную. 
      • Единица измерения. Элемент справочника "Единицы измерения".
      • Коэффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
