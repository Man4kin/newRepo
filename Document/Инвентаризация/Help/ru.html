Документ "Инвентаризация"

Общее

Документ "Инвентаризация" предназначен для проведения инвентаризации на оптовых, розничных складах и в неавтоматизированных торговых точках. Инвентаризация в неавтоматизированной торговой точке ведется в отпускных (розничных) ценах.
Инвентаризация производится отдельно по каждому складу. Проведение документа позволяет привести в соответствие остатки товаров в системе с реальными остатками на складе.
Документ "Инвентаризация" формирует сличительную ведомость и инвентаризационную опись при проведении инвентаризации на складах фирмы, а также выписки актов списания и оприходования излишков на основании данного документа при наличии расхождений между фактическими и документально подтвержденными остатками номенклатурных позиций.
Данным документом не только отражается факт проведения плановой инвентаризации товаров, но так же и осуществляется текущая корректировка товарных остатков. Например, с целью коррекции отрицательных остатков по партиям прихода.

Форма шапки документа

При добавлении нового документа открывается окно "Инвентаризация".
Номер, дата и автор указываются в шапке документа автоматически.

В шапке документа вручную необходимо заполнить с следующие поля:

      • Фирма, в которой проводится инвентаризация.  Элемент справочника "Фирмы".
      • Склад, на котором проводится инвентаризация . Заполняется из выпадающего списка: главный склад, торговый зал.
      • Основание. Основание проведения инвентаризации - заполняется вручную.

Флаг "Инвентаризация в разрезе фирм" . Инвентаризация может проводиться как по складу (флаг должен быть снят), так и в разрезе фирм (флаг  должен быть установлен).

Флаг "Использовать только табличную часть"  указывает на то, что инвентаризация проводится только по тем товарам, которые указаны в табличной части документа. 

ВАЖНО! При снятиии флага "Использовать только табличную часть"  остатки товаров, которых нет в таличной части документа будут удалены после проведения документа.

Табличная часть документа

Табличная часть документа содержит вкладки:

          • Состав
          • Пересортица

Вкладка "Состав".

На данной вкладке  заполняются следующие реквизиты:

              • Номенклатура. Элемент справочника "Номенклатура".
              • Характеристика. Заполняется из справочника "Номенклатура", если для данного товара ведется учет по дополнительным реквизитам.
              • Цена товара .
              • Кол. факт.  Фактическое количество товара на складе. Заполняется вручную.
              • Сумма факт. Сумма, в которую оценивается фактическое количество товара на складе.
              • Кол. учет.  Количество товара на складе по данным бухгалтерского учета.
              • Сумма учет. Сумма, в которую оценивается количество товара на складе по данным бухгалтерского учета.

При нажатии кнопки "Заполнить" табличная часть документа заполняется товарными  остатками данного склада (остатки при этом не равны нулю) либо всеми товарными позициями, которые хранятся в справочнике "Номенклатура".
Остатки вводятся по цене: последней закупочной либо средневзвешенной (себестоимость остатка товаров,  деленная на остаток товара). В том случае, если  заполняете табличную часть остатками по последней цене закупки, то отбор закупочных цен осуществляется по той фирме, которая указана в шапке документа в реквизите «Фирма».

Важно! При заполнении документа  с использованием кнопки "Заполнить" все данные, ранее введенные в табличной части вручную, будут удалены!

С помощью кнопки "Обновить док. данные"   можно обновить данные уже заполненного документа по остаткам товаров и ценам в том случае, если  документ еще не проведен, а данные по остаткам уже изменились.

В случае, если флаг "Инвентаризация в разрезе фирм"  установлен, в документе появляется вторая табличная часть, в которой необходимо заполнить следующие колонки:

        • Фирма, по которой проводится инвентаризация. Элемент справочника "Фирмы".
        • Цена товара. Заполняется вручную.
        • Кол. факт. Фактическое количество товара на складе. Заполняется вручную.
        • Сумма факт. Сумма, в которую оценивается фактическое количество товара на складе.
        • Кол. учет.  Количество товара на складе по данным бухгалтерского учета.
        • Сумма учет. Сумма, в которую оценивается количество товара на складе по данным бухгалтерского учета.

С помощью кнопки "Очистить" можно удалить даные из всех такбличных частей.

Вкладка "Пересортица".

Вкладка предназначена для исправления в системе пересортицы, возникшей по товару. Документ фиксирует факт переноса некоторого количества товара из одного реквизита  дополнительного учета в другой.

ВАЖНО! Пересортица составляется только для тех товаров, по которым ведется дополнительный учет.

Данную вкладку можно заполнить как автоматически, используя кнопку "Заполнить", так и вручную

В табличной части вкладки необходимо заполнить следующие реквизиты:

            • Номенклатура (излишек). Товар, по которому обнаружен излишек. Заполняется подбором из справочника "Номенклатура".
            • Характеристика. Группа дополнительного  учета товара, по которому обнаружен излишек. Заполняется из справочника "Номенклатура",
            • Номенклатура (недостача). Товар, по которому обнаружена недостача.
            • Характеристика. Новая группа дополнительного  учета товара, по которому обнаружена недостача. Заполняется из справочника "Номенклатура",
            • Количество  товара.

В случае, если флаг "Инвентаризация в разрезе фирм"  установлен,  на вкладке появляется вторая табличная часть, содержащая те же колонки, что и первая табличная часть, при этом добавляется колонка "Фирма" (элемент справочника "Фирмы".  )
