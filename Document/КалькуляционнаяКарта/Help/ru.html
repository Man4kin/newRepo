Документ "Технологическая карта"

Общее

Конфигурация дает возможность для автоматизации собственного производства, построенную по схеме, используемой в общественном питании.
Для этого используются  документы "Технологическая карта"  и  "Готовая продукция".

Форма шапки документа

При добавлении нового документа открывается окно "Технологическая карта".
Номер, дата и автор указываются в шапке документа автоматически.


В шапке документа "Технологическая карта"  содержатся следующие реквизиты:

      • Продукция. Элемент справочника "Номенклатура".
      • Вид производства. Приготовление или разделка - выбирается из списка.
      • Количество порций. Заполняется вручную.
      • Номер по стравочнику. Заполняется вручную.


Табличная часть документа

В табличной части документа необходимо заполнить следующие колонки:
      • Номенклатура. Элемент справочника "Номенклатура".
      • Характеристика. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
      • Количество. Заполняется вручную.
      • Количество нетто. Заполняется вручную.
      • Единица измерения. Элемент справочника "Единицы измерения".
      • Коээффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
      • Технологическая карта. Заполняется из списка уже существующих в базе технологических карт.
