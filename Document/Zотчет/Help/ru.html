Документ "Z-отчет"

Общее

«Z-отчеты» предназначены для оформления операций, связанных с продажей в розницу товаров, как купленных, так и комиссионных, а также для оформления операций, связанных с возвратом товара.
«Z-отчет» используется для возврата ранее купленного товара, если возврат происходит в день покупки, во всех остальных случаях используется возвратная накладная
Документ "Z-отчет"создается автоматически по результатам работы POS-терминала, либо заполняется кассиром или другим сотрудником вручную по сводным результатам продаж через кассу.
«Z-отчет» – товарный документ, предназначенный для оформления реализации следующих ценностей:

      • Товаров (весового или штучного)
      • Наборов
      • Продукции собственного производства
      • Услуг
      • Тары

Форма шапки документа

При добавлении нового документа открывается окно "Zотчет" .
Номер, дата и автор указываются в шапке документа автоматически. 
В шапке документа  необходимо заполнить поле "Фирма" (элемент справочника "Фирмы"). Указывается фирма, от лица которой ведутся взаиморасчеты.

Флаг «Запрет модификации системой SMS» устанавливается автоматически при загрузке данных о продажах из модуля SMS и запрещает перезапись документа в случае изменения  данных о продажах в модуле (например, осуществлялась повторная загрузка данных о продажах из кассовых комплексов в модуль SMS).

Важно! Ввести документ можно как автоматически, так и вручную, используя бумажный документ Z-отчет, полученный на кассовом аппарате.

 

Табличная часть документа

В документе содержится две табличные части.

В первой табличной части необходимо заполнить следующие колонки:

          • ПОС. Кассовый комплекс, используемый при розничных продажах. Элемент справочника "ККМ".
          • Смена. Номер смены кассира.
          • Смена SMS. Заполняется автоматически в случае использования встроенного модуля SMS (Sales Management System) - системы управления продажами.
          • Дата смены. По умолчанию автоматически ставится текущая дата (можно редактировать вручную).
          • Сумма выручки. Сумма  по Z-отчету, заполняется вручную
          • В т.ч. наличных.  Сумма наличных денег, находящаяся в ККМ на момент снятия z-отчета.
          • Свободная сумма. Сумма денежных средств, которая не связана ни с какими конкретными проданными товарами. Свободная сумма обычно образуется в качестве платы за товары, поштучный учет которых на складах не ведется.

            Вторая табличная часть содержит вкладки:
              • Состав.
              • Кредитные карты
              • Оплата мобильной связи

Вкладка "Состав".

На данной вкладке необходимо заполнить следующие реквизиты:

        • Номенклатура.  Выбор товаров из справочника "Номенклатура".
        • Характеристика/партия. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
        • Количество. Проданное количество согласно фискальным документам.
        • Единица измерения. Элемент справочника "Единицы измерения".
        • Коэффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
        • Ставка НДС. Автоматически указывается ставка, указанная в карточке товара. Возможно редактирование при помощи справочника  "Ставки НДС".
        • Сумма НДС. Рассчитывается автоматически.
        • Сумма. Сумма продажи по фискальным документам.
        • Склад, с которого производится отгрузка (продажа) товара. Заполняется из списка складов.
        • Состав набора. Заполняется только для товаров с типом "набор" с помощью документа "Состав набора" .


Вкладка "Кредитные карты".

На данной вкладке необходимо заполнить следующие реквизиты:

            • Тип карты. Тип кредитной карты, по которой осуществлялась розничная продажа, например, Visa.  Элемент справочника "Кредитные карты".
            • Сумма. Сумма, оплаченная по кредитной карте. Заполняется вручную.

Вкладка "Оплата мобильной связи".

На данной вкладке заполняются следующие колонки:

            • Мобильная связь/тип оплаты. Здесь автоматически отображаются все виды мобильной связи (Би Лайн, МТС, Мегафон) и виды оплаты (оплата наличными и по кредитным картам). Кредитные карты, принимаемые к оплате, вводятся пользователем в справочник "Кредитные карты".
            • Сумма. Сумма оплаты мобильной связи.
