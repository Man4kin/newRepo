Документ "Расходная накладная"

Общее

Расходные накладные предназначены для оформления операций, связанных с продажей оптом или передачей ТМЦ на реализацию контрагентам. Накладная соответствует реализации следующих ценностей:

      • товаров (весового или штучного)
      • услуг
      • наборов
      • продукции собственного производства


Расходную накладную можно вводить на основании счета или приходной накладной.

Форма шапки документа

При добавлении нового документа открывается окно "Расходная накладная".
Номер, дата и автор указываются в шапке документа автоматически.

В шапке документа содержатся следующие реквизиты:

      • Фирма. Элемент справочника "Фирмы".
      • Склад. Заполняется из списка складов.
      • Контрагент. Элемент справочника "Контрагенты".
      • Договор. Элемент справочника "Договоры".
      • Основание. Заполняется вручную.
      • Категория цен. Элемент справочника "Категории цен". Например: закупочная, оптовая, розничная.
      • Вариант расчета НДС. Заполняется из выпадающего списка: НДС сверху; НДС в том числе; без НДС.

Табличная часть документа

Табличная часть документа содежит вкладки:

      • Состав
      • Дополнительные сведения
      • Закрытия обязательств

Вкладка "Состав"

На данной вкладке необходимо заполнить следующие реквизиты:

            • Номенклатура. Элемент справочника "Номенклатура". Добавить товар можно  с помощью кнопки "Добавить", либо с помощью опции "Подбор".
            • Характеристика. Заполняется из справочника "Номенклатура" в случае, если для данного товара ведется учет по дополнительным реквизитам.
            • Количество. Заполняется вручную.
            • Единица измерения. Элемент справочника "Единицы измерения".
            • Коэффициент пересчета единицы измерения товара в базовую. Заполняется автоматически.
            • Ставка НДС. Элемент справочника "Ставки НДС".
            • Цена. Заполняется вручную.
            • Сумма. Рассчитывается автоматически.
            • Сумма НДС. Рассчитывается автоматически.
            • Всего. Рассчитывается автоматически.
            • Состав набора. Автоматически заполняется для товаров с типом "набор".

Вкладка "Дополнительные сведения" 

На данной вкладке необходимо заполнить следующие реквизиты:

        • Грузополучатель. Элемент справочника  "Контрагенты".
        • Адрес доставки. Заполняется вручную.
        • Грузоотправитель. Тип данных выбирается из списка: контрагенты или фирмы. Далее указывается элемент соответствующего справочника.


Вкладка "Закрытие обязательств" 

На вкладке "Закрытие обязательств"  необходимо заполнить следующие реквизиты:

На данной вкладке необходимо заполнить следующие колонки:

        • По документу.
        • Валюта.
        • Ставка НДС.
        • Сумма валютная.
        • Сумма.
        • Сумма НДС.

Меню вкладки "Закрытие обязательств" содержит следующие кнопки:

        • Подбор. При использовании этой кнопки открывается окно "Подбор обязательств".
        • FIFO. Кнопка осуществляет автоматический отбор обязательств в табличную часть для погашения по одноименному методу  (first in – first out (первый пришел – первый ушел).
        • Очистить. При нажатии этой кнопки все данные из табличной части будут удалены.
