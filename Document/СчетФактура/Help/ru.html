Документ "Счет-фактура"

Общее

Документ предназначен для выписки счета-фактуры при реализации товаров, работ и услуг.

Данные из счетов-фактур составляют содержимое "Книги продаж" – отчета формируемого по желанию пользователя.

ВАЖНО! Документ вводится только на основании проведенного документа реализации, в котором уже есть все основные данные: о покупателе, о реализованных товарах, работах или услугах.

 

Форма шапки документа

При добавлении нового документа открывается окно "Счет-фактура".
Номер, дата и автор указываются в шапке документа автоматически.

Далее пользователем заполняются следующие реквизиты:

        • Фирма, от лица которой ведутся взаиморасчеты. Элемент справочника "Фирмы"
        • Вариант расчета НДС. Заполняется выбором из выпадающего списка: НДС сверху, НДС в том числе, без НДС.
        • Контрагент, с которым проводятся взаиморасчеты. Элемент справочника "Контрагенты"
        • Договор с контрагентом. Элемент справочника "Договоры"
        • Грузоотправитель. Грузоотправителем может быть как контрагент, так одна из наших фирм.  Из предложенного списка пользователь выбирает нужный справочник, а затем его элемент.
        • Грузополучатель.  Заполняется при помощи справочника "Контрагенты" .
        • Основание.  Автоматически указывется номер и дата расходной накладной, на основании которой введен документ "Счет-фактура".


Табличная часть документа

 

Табличная часть документа автоматически заполняется данными из документа-основания (расходной накладной) и содержит следующие реквизиты:

        • Номенклатура
        • Характеристика
        • Количество
        • Единица измерения
        • Коэффициент
        • Ставка НДС
        • Сумма НДС
        • Цена
        • Сумма

 

Кнопка «Переоформить» служит для переоформления счета-фактуры в том случае, если она была введена на основании накладной, которая была впоследствии изменена.
